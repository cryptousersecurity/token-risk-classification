
# TSB-2023-006 SelfDestruction
## Description

The contract can be destroyed, resulting in the loss of all functionality and zeroing of assets.

## Risk Pattern

```solidity
function close(address payable to) external onlyOwner { 
  selfdestruct(to); 
}
```
