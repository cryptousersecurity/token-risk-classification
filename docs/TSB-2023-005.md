
# TSB-2023-005 HiddenOwnership
## Description

To hide the status of a privileged address, set it to an unreadable address, or to name it using a non-standard method.

## Risk Pattern

```solidity
modifier superman() {
  require(superman == _msgSender(), );
}
```
